แบบทดสอบ บทที่ 9
1. ให้ระบุว่า Component ต่อไปนี้ จัดเป็น Component ประเภทใด
    a) โปรแกรม MSWORD.EXE
    b) Table ต่างๆ ใน Database
    c) เอกสารที่ถูกสร้างขึ้นจากโปรแกรม Microsoft Word
    d) Dynamic Link Library (DLL)
    e) แฟ้มข้อมูลที่เก็บรายละเอียดราย Record ของการขายสินค้า

2. ให้ออกแบบ Component Diagram และ Deployment Diagram สำหรับระบบการขายปลีกสินค้า รวมถึงการวาง Component ลงบน Node ต่างๆตามเงื่อนไขต่อไปนี้
“ผู้ใช้งานต้องการให้ระบบทำงานบนระบบปฏิบัติการ Windows ที่มีการติดต่อเครือข่ายข้องถิ่น (LAN) ทั้งนี้ ผู้ใช้งานจะต้องการให้ระบบ Relational Database สำหรับการจัดเก็บข้อมูลต่างๆ ส่วนหน้าจอใช้งานต่างๆ ไม่จำเป็นต้องใช้ Web-based Application นอกจากนี้ ผู้ใช้งานยังต้องการให้มีการ Printer เป็นส่วนกลางเพื่อการพิมพ์งานของระบบทั้งหมด”

3. ให้ออกแบบ Component Diagram และ Deployment Diagram สำหรับระบบ Book Store Services System รวมถึงการวาง Component ลงบน Node ต่างๆตามเงื่อนไขต่อไปนี้
“เนื่องจากคลังหนังสือกับร้านหนังสือ อยู่คนละอาคารซึ่งอยู่กันประมาณ 2 กิโลเมตร โดยส่วนของระบบที่ดูแลการ ซื้อ-ขายและยืมหนังสือ จะทำงานอยู่ที่ร้าน แต่ระบบในส่วนของการสั่งซื้อหนังสือและอื่นๆ จะทำงานอยู่ที่สำนักงานที่คลังหนังสือ”
