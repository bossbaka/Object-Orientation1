แบบทดสอบ บทที่ 2
1. ให้ระบุว่าใจความสำคัญของคำหรือข้อความต่อไปนี้ กล่าวถึงสิ่งใด ระหว่าง Object, Class, Attribute หรือ Method
    ก. นายแดงที่อาศัยอยู่ที่เขตดินแดง มีเลขที่บัตรประชาชน 3-1499-00456-87-5
    ข. คนไทย
    ค. สถานะของสัญญาณไฟจราจร
    ง. การให้บริการ
    จ. ระดับของความเสียหายที่เกิดจากแผ่นดินไหว
    ฉ. การทำงานของเครื่องจักรในโรงงานทอผ้า
    ช. ผลของการสอบ
    ซ. การเคลื่อนที่ของสัตว์เซลเดียว

2. ข้อเท็จจริงจากประโยคต่อไปนี้ สามารถอธิบายได้ด้วย Abstraction ประเภทใด
    ก. คนมีอวัยวะต่างๆ คือ แขน ขา ลำตัว และศีรษะ
    ข. นายดำ และนายแดง เป็นผู้ชาย นางสาวขาว และนางสมศรี เป็นผู้หญิง
    ค. โรงพิมพ์เป็นผู้ผลิตหนังสือ
    ง. คนเป็นเจ้าของคอมพิวเตอร์
    จ. สัตว์ปีกมีหลายประเภท ได้แก่ ไก่ นก และ นกน้ำ
    ฉ. คอมพิวเตอร์จัดเป็นเครื่องคำนวณประเภทหนึ่ง
    ช. นางสาว ข เลี้ยงสุนัข
    ซ. โรงเรียนประกอบด้วยอาคารหลายๆ อาคาร แต่ละอาคารประกอบด้วยห้องหลายๆ ห้อง

3.



    จากรูปข้างต้น Class "Membership" และ Method "getRight" ทำหน้าที่อ่านค่าของ Attribute "right" และ Method "setRight" ทำหน้าที่กำหนดค่าของ Attribute "right"
    สำหรับ Class "PremiumMembeship" มี Method "getPremiumLevel" ทำหน้าที่อ่านค่าของ Attribute "premiumLevel" ในขณะที่ Method "setPremiumLevel" ทำหน้าที่กำหนดค่าของ Attribute "premiumLeve"
    ตอบคำถามต่อไปนี้
    ก. Attribute ใดบ้างของ Object ของ Class "Membership" ที่สามารถถูกเข้าได้โดยตรงจากภายนอก
    ข. Attribute ใดบ้างของ Object ของ Class "Membership" ที่สามารถถูกเข้าได้โดยตรงจากภายนอก ทั้งทางตรงและอ้อม
    ค. Attribute ใดบ้างของ Object ของ Class "Membership" ที่ไม่สามารถถูกเข้าถึงได้เลยจากภายนอก
    ง. จงวาด Internal View ของ Class "PremiumMembership" และ "GoldClassMembership"

4. ให้เขียน Internal View ของ Class "SeaAnimal และ Internal View ของ Class "ComboPlayer"

5. Generalization Abstraction และ Aggregation มีความเหมือนหรือต่างกันอย่างไร

6. Aggregation Abstraction และ Association Abstraction มี ความเหมือนหรือต่างกันอย่างไร แทนกันได้หรือไม่ อย่างไร

7. ให้อธิบายความแตกต่างระหว่าง Aggregation Abstraction และ Composition
