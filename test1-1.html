แบบทดสอบ บทที่ 1
1. ให้แยกแยะว่าสิ่งต่อไปนี้ สิ่งใดคือ Class สิ่งใดคือ Object
   หนังสือ  ชุดว่ายน้ำ  คอมพิวเตอร์คอมแพ็ค  รถยนต์ยี่ห้อ Toyota
   แว่นตา  นาย ก  ดินสอสีดำที่อยู่ในกระเป๋าของเด็กชาย ข

2. จงอธิบายความแตกต่างระหว่าง Class กับ Data Type

3. จากสถานการณ์ต่อไปนี้ ให้แยกแยะว่ามี Object อะไรบ้าง และ Function ที่ Object แต่ละตัวต้องมีคืออะไร
